# Security Policy

AdaSDF-CL is currently an alpha / research-preview project. It is not an
industrial safety-certified collision library.

## Reporting Security Issues

Security contact: to be added before public release.

Until a dedicated security contact is published, please report non-sensitive
security concerns through the GitHub issue tracker after the public repository
exists. Do not post sensitive, proprietary, closed-source, or embargoed model
data in a public issue.

## Model and Asset Handling

AdaSDF-CL does not adjudicate third-party model or asset copyright. Users are
responsible for ensuring they have the right to share any meshes, `.sdfbin`
assets, logs, or reproduction files they provide.

Do not upload sensitive or closed-source models to public issues. Prefer small
project-generated reproducers.

## Current Scope

This project currently provides research-preview code and documentation only. It
does not provide industrial safety certification, security hardening guarantees,
or safety-critical deployment guidance.
