# Code of Conduct

AdaSDF-CL is intended to be a respectful research and engineering project.

## Expected Behavior

- Be kind and constructive.
- Discuss technical disagreements with evidence and patience.
- Credit prior work.
- Document limitations honestly.
- Avoid hostile, discriminatory, or harassing behavior.

## Unacceptable Behavior

- Harassment or personal attacks.
- Discriminatory language.
- Publishing private information without permission.
- Misrepresenting research-preview behavior as certified production capability.

## Reporting

For now, report conduct concerns through the project issue tracker or maintainer contact listed by the eventual public repository. If no public repository has been assigned yet, keep reports private until a maintainer channel is available.
