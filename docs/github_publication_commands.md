# GitHub Publication Commands

Use `v1.1.0-alpha` for the expanded-SDF accuracy audit pre-release.

The original v1.0.2-alpha, v1.0.2-alpha.1, and v1.0.3-alpha tags are retained
for traceability. The recommended public pre-release is v1.1.0-alpha.

```bash
git remote -v
git push origin main
git push origin v1.1.0-alpha
```

Do not move or force-push older alpha tags.
