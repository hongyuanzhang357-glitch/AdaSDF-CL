# GitHub Publication Commands

Use `v1.0.2-alpha.1` for the CUDA benchmark semantics and CI portability
hotfix pre-release.

The original v1.0.2-alpha tag is retained for traceability. The recommended
public pre-release is v1.0.2-alpha.1.

```bash
git remote -v

```

Do not move or force-push older alpha tags.
