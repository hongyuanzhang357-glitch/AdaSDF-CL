# GitHub Publication Commands

Use `v1.0.3-alpha` for the CUDA performance optimization pre-release.

The original v1.0.2-alpha and v1.0.2-alpha.1 tags are retained for
traceability. The recommended public pre-release is v1.0.3-alpha.

```bash
git remote -v
git push origin main
git push origin v1.0.3-alpha
```

Do not move or force-push older alpha tags.
