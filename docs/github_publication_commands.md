# GitHub Publication Commands

Use `v1.3.0-alpha` for the mesh readiness pre-release.

The original v1.0.2-alpha, v1.0.2-alpha.1, v1.0.3-alpha, v1.1.0-alpha, and
v1.1.1-alpha and v1.2.0-alpha tags are retained for traceability. The
recommended public pre-release is v1.3.0-alpha.

```bash
git remote -v
git push origin main
git push origin v1.3.0-alpha
```

Do not move or force-push older alpha tags.
